# pixelize

## Install
```
$ pip install -r requirements.txt
$ npm install
```

## Create the image
An SVG image like [this](https://github.com/daiiz/pixelizer/blob/master/html/pancake.a.svg) will be generated to `html/`.
```
$ python svg_c.py raw/pancake.jpg
```

## Preview
```
$ open ./html/view.html
```
